# cloud-itonami-iso3166-rou

Open ISO 3166 Blueprint for **ROU**: Romania.

Independent public-sector market-entry & procurement-compliance service
for an already-incorporated operator entering public contracts in Romania.

## Official surface

- Procurement: SEAP / SICAP electronic public procurement system
- Business/tax: ONRC commercial register + CUI tax ID
